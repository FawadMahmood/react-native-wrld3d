# react-native-wrld3d
This is a bridge for wrld3d maps for React Native
## Installation

```sh
npm install react-native-wrld3d
```

## Usage

```js
import { Wrld3dView } from "react-native-wrld3d";

// ...

<Wrld3dView color="tomato" />
```
